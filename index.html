Featured thesis project

WordFlow - AI Chrome Extension for Dyslexic Writing Support

WordFlow is my final-year thesis project: an AI-powered Chrome extension released as a browser tool for real-time writing correction, built for dyslexic users and evaluated against established writing support workflows.

Chrome Extension Final-year thesis Privacy-first

Built from model to browser.

I designed the full workflow: a fine-tuned T5 transformer model, a stateless Python FastAPI backend, and a Manifest V3 JavaScript extension that helps users correct writing in real time.

  • Custom 6,571-pair dataset combining curated, JFLEG benchmark and synthetic dyslexic error patterns.
  • 78% full correction accuracy, around 92% phonetic error accuracy, and 300-800ms CPU latency.
  • No raw user text stored, HTTPS encrypted flow, and a GDPR-aware architecture.
  • Real-user evaluation through Dyslexia Ireland participants and comparison with Grammarly.
